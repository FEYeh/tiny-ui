### With Icon

Using an icon replaces a text or combination.

<!--start-code-->

```jsx
render(
  <Breadcrumb>
    <Breadcrumb.Item>
      <Icon type="home" />
    </Breadcrumb.Item>
    <Breadcrumb.Item>
      <a href="#/components/breadcrumb"><Icon type="inspection" />Project List</a>
    </Breadcrumb.Item>
    <Breadcrumb.Item>A Project</Breadcrumb.Item>
  </Breadcrumb>
);
```

<!--end-code-->
