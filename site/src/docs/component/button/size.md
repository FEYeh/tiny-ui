### Size

There are three different sizes, `lg`, `md`, `sm`. The default size is `md`.

<!--start-code-->

```jsx
render(
  <>
    <Button color="primary" size="lg">Large Size</Button>
    <Button color="primary">Default Size</Button>
    <Button color="primary" size="sm">Small Size</Button>
  </>
);
```
